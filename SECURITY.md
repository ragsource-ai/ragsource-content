# Sicherheitsrichtlinie / Security Policy

## Inhalt dieses Repos / About this Repository

Dieses Repository enthält ausschließlich **öffentliche Rechtstexte** (Gesetze, Satzungen, Verordnungen) nach § 5 UrhG oder vergleichbar. Es enthält keinen ausführbaren Code.

This repository contains only **public legal texts** (laws, statutes, ordinances). It contains no executable code.

---

## Falsche oder irreführende Inhalte / Incorrect or Misleading Content

Wenn du einen **inhaltlichen Fehler** in einem Rechtstext findest (veraltete Fassung, falsche Paragraphen, fehlendes Frontmatter), öffne bitte ein **GitHub Issue** oder einen **Pull Request**.

If you find a **content error** in a legal text (outdated version, wrong paragraphs, missing frontmatter), please open a **GitHub Issue** or **Pull Request**.

---
